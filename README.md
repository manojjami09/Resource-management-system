# Resource Manager

A comprehensive resource management application consisting of a React frontend and a Spring Boot backend.

## Project Structure

- `/frontend`: The React application (Vite, TypeScript, TailwindCSS). It handles the user interface, routing, and data presentation.
- `/backend`: The Java Spring Boot application. It serves as the core API, managing data persistence in PostgreSQL, authentication, and AI-driven insights.
- `/legacy-backend`: The legacy Node.js/Express backend preserved for API shape reference.
